# рҹҡҖ Launchpad

A modern macOS application launcher built with SwiftUI and TCA (The Composable Architecture). Inspired by macOS Launchpad and LaunchOS.

![macOS](https://img.shields.io/badge/macOS-14.0+-blue.svg)
![Swift](https://img.shields.io/badge/Swift-5.9-orange.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

## вңЁ Features

- **рҹ“Ҡ Application Grid** - Beautiful grid layout displaying all your applications
- **рҹ”Қ Instant Search** - Quickly find apps by name
- **рҹ“Ғ Smart Folders** - Organize apps into custom folders with drag-and-drop
- **рҹ”„ Drag & Drop** - Rearrange apps and create folders by dragging
- **рҹ“Ӯ Multi-Directory Scanning** - Scans `/Applications`, `/System/Applications`, `~/Applications`, and more
- **рҹ’ҫ Persistent Storage** - Folders and arrangements are saved automatically
- **рҹҺЁ Native UI** - Blends seamlessly with macOS using native blur effects
- **вҢЁпёҸ Keyboard Shortcuts** - ESC to clear search, context menu support

## рҹӣ  Installation

### Quick Install (Recommended)

```bash
# Clone the repository
git clone https://github.com/yourusername/Launchpad.git
cd Launchpad

# Build and install
./install.sh
```

### Manual Install

```bash
# Build the app
./build-app.sh

# Copy to Applications
cp -r build/Launchpad.app /Applications/
```

### Run from Source

```bash
# Build and run
swift build
.build/debug/Launchpad

# Or open in Xcode
open Package.swift
```

## рҹ“– Usage

### Basic Operations

| Action | How To |
|--------|--------|
| Launch App | Click on the app icon |
| Search Apps | Type in the search bar |
| Clear Search | Press `ESC` key |
| Refresh Apps | Click refresh button in toolbar |

### Folder Management

| Action | How To |
|--------|--------|
| Create Folder | Drag one app onto another app |
| Create Empty Folder | Click folder+ button in toolbar |
| Add App to Folder | Drag app onto a folder |
| Open Folder | Click on the folder |
| Rename Folder | Open folder вҶ’ Click pencil icon |
| Remove App from Folder | Right-click app вҶ’ Remove from Folder |
| Delete Folder | Right-click folder вҶ’ Delete Folder |
| Exit Folder | Press `ESC` or click back button |

### Edit Mode

Click the `вҮ…` button in the toolbar to enter edit mode:

- Drag apps to rearrange them
- Drag apps onto each other to create folders
- Drag apps onto folders to add them
- Click the checkmark button to exit edit mode

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `ESC` | Clear search / Exit folder |
| `Cmd+F` | Open search window |
| `Cmd+,` | Open settings |
